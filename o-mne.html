O mne

Jmenuju se holleris a tohle je misto kde bych mel psat neco chytryho o sobe takze to vezmem zkratkou.

Programovani me bavi proto ze pocitac si nestezuje nehraje hry nedela drama a kdyz neco rozbiju tak vim ze je to moje vina.

Spoluprace s pocitacem je jednoducha funguje nebo nefunguje a na rozdil od holek mi po trech dnech neprestane psat bez vysvetleni.

Ucim se HTML CSS JavaScript Python C# a C++ protoze porad nevim kterej jazyk me zklame omrzi jako prvni.
